# Vue Componentized Calendar

Более правильная версия таймтрекера на Vue 3 + Vite.

Что уже перенесено:
- авторизация через Supabase magic link
- недельный календарь
- 1 или 2 задачи в ячейке
- half-cell (0.25ч)
- переключение рабочего/выходного дня
- проекты, группы, бюджет, комментарий, цвет, архив/оплачено
- редактирование названия проекта на месте

## Запуск

```bash
npm install
npm run dev
```
